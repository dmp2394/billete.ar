# Billete.ar

TP de Programacion II

## Requisitos

- Java 9 o superior
- Eclipse IDE
- Git

## Cómo clonarlo
En el directorio que se desee bajar los archivos, ejecutar:
```bash
git clone https://github.com/dmp2394/billete.ar.git
```

## Importar en Eclipse

1. Abrí Eclipse
2. **File → Import → General → Existing Projects into Workspace**
3. Seleccioná la carpeta donde clonaste el proyecto
4. Click en **Finish**

## Mantenerlo actualizado

Cuando haya cambios nuevos, desde la terminal:

```bash
git pull
```

O desde Eclipse: click derecho en el proyecto → **Team → Pull**

## Impactar cambios

```bash
git add .
git commit -m "descripción"
git pull        # traer cambios de tus compañeros
git push        # subís los tuyos
```
